Client x Server

Voltar

Client: São dispositivos como Notbooks, Computadores, celulares, que exige determinados serviços a um servidor,dependem de acesso ao servidor para ler e salvar nele as informações de todos trabalhos feitos no programa. Quando acessamos um site pelo navegador, uma requisição é disparada pela internet para o servidor no qual esse site está hospedado e retorna com a informação solicitada, ou seja, os clients iniciam sessões de comunicação com os servidores que aguardam requisições de entrada.

Server: Retorna a resposta solicitada pelos clientes em HTML, CSS, javascrip, arquivos estaticos e outros.

O que é HTML?

Voltar

Em inglês HyperText Markup Language, significa Linguagem de Marcação de Hipertexto. São marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento. O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, Os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.
